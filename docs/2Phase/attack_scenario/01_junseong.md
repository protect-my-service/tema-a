##### ATK-F01 — 공격 흐름 요약

| 단계 | 행동                                 | 정상 방어 기대         | 실제 결과                    |
|----|------------------------------------|------------------|--------------------------|
| 1  | 동일 상품에 `quantity=2147483647` 반복 추가 | 큰 수량 또는 누적 한도 차단 | 요청 정상 처리                 |
| 2  | 같은 상품을 6회 누적                       | `int` 오버플로우 차단   | `cart_item.quantity=-6`  |
| 3  | 장바구니 조회                            | 총액은 0 이상 유지      | `totalAmount=-213499.74` |

대표 요청은 아래 형태다. 같은 상품에 반복 실행해 음수 수량을 만든다.

```bash
POST /api/v1/cart/items
X-Member-Id: 4
{"productId":3,"quantity":2147483647}
```

##### ATK-A01 — 공격 흐름 요약
